# 🔨 Funcionalidades do projeto

`Funcionalidade 1`: Adicionar usúarios do Github na Tabela.

`Funcionalidade 2`: Remover favoritos da tabela.

`Funcionalidade 3`: Deixar a tabela sem nenhum favorito.
